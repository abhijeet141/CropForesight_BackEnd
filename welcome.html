Welcome to the Crop Prediction API

To use the /predict API, send a POST request to the endpoint with the following JSON data: 

            Endpoint: /predict
            Method: POST
            Content-Type: application/json
            Body:
            {
                "cropInfo": {
                    "nitrogen": [Integer],
                    "phosphorus": [Integer],
                    "potassium": [Integer],
                    "temperature": [Integer],
                    "humidity": [Integer],
                    "ph": [Integer],
                    "rainfall": [Integer]
                }
            }

Replace [Integer] with the respective values for your crop prediction.

Example Request

                Endpoint: /predict
                Method: POST
                Content-Type: application/json
                Body:
                {
                    "cropInfo": {
                        "nitrogen": 50,
                        "phosphorus": 25,
                        "potassium": 30,
                        "temperature": 28,
                        "humidity": 70,
                        "ph": 6.5,
                        "rainfall": 100
                    }
                }

Example Response

            Status: 200 OK
            Content-Type: application/json
            Body:
            {
                "prediction": "Your crop prediction result will be shown here."
            }

Note: Make sure to replace the example values with your actual data when making requests to the API.
